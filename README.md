# GesturePlay 🎮

An AI-powered gesture-controlled gaming platform that allows users to play games using hand gestures detected through a webcam.

## Tech Stack

- React
- FastAPI
- OpenCV
- MediaPipe
- TensorFlow (later)

## Features

- [ ] Webcam Integration
- [ ] Hand Detection
- [ ] Gesture Recognition
- [ ] Browser Game
- [ ] AI Model Training
- [ ] Real-Time Gesture Control
